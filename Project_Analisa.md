## Status Saat Ini

BOS dapat menghasilkan:

* BULLISH
* BEARISH
* SIDEWAYS

CHOCH dapat menghasilkan:

* BULLISH
* BEARISH
* NONE

NONE dianggap normal jika belum ada perubahan struktur.

## Layout Saat Ini
